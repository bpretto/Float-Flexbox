Flexbox

Propriedades Flexbox utilizadas para desenvolvimento dos layouts:

Display: define o elemento como um container de itens flex;


Order: controla a ordem do conteúdo;


Height/Width: Não é necessário passar ambas as propriedades para todos os elementos, visto que as Flexboxes herdam tais propriedades de seus contêineres;
